How this works

This tool runs vendor invoice downloads one-by-one in the background, tracks progress live, and sends a single batch email when the run finishes.
  • Batch run downloads every configured account across vendors.
  • Single run downloads one specific vendor/account from “Quick actions”.
  • Email override applies only to the current run unless you save it as the default.
  • Results show success/failure for each account with filenames or error messages.
